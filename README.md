> ## Toplama

```python
print(3+5)
```

> ## Çıkartma
```python

```

> ## Çarpma 
```python
print(3*5)
```

> ## Bölme
```python
print(10/2)
[out] 5.0 
```

> ## Üstlü
```python
print(2**3)
[out] 8 
```

> ## Mod  
```python
print(10%3)
[out] 1
```

> ## Örnekler
```python
KullanıcıYasi=15    #Kullanıcı adını tanımlıyoruz
print(KullanıcıYasi * 3)
[out] 45
```

```python
yari_cap=10
dairein_cevresi=2*3.14*yari_cap  #Yarı çapı yukarda tanımladığımız için direkt yarı çap diyebiliriz
print(dairenin_cevresi)
[out] 62.80000004
```
